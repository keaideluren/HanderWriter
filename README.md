# HanderWriter
# 手写演示
比较流畅的，可以空心实心，辅助是指展示绘制时使用的点，横线长度是按压力度的大小，各色的点是贝赛尔曲线的控制点，每一个触摸点都有模拟压力大小和线宽计算的四个控制点。
